## Proyectos Destacados

### 1. Prediccion de Precios de Casas en Uruguay
**Machine Learning | Regresion | CatBoost**

Modelo predictivo para estimar precios de propiedades inmobiliarias en Uruguay utilizando datos de Mercado Libre y El Gallito.

- **Dataset**: 177,394 observaciones, 33 variables
- **Modelo Final**: CatBoost optimizado con Optuna
- **Resultado**: MAPE de 25% para Montevideo (~USD 50,000 de error tipico)
- **Tecnologias**: Python, Pandas, Scikit-learn, CatBoost, Optuna

[Ver Proyecto Completo](projects/prediccion-casas/README.md)

---

### 2. From Data to Goal - Prediccion de Partidos de Futbol
**Machine Learning | Clasificacion Multiclase | Sports Analytics**

Sistema de prediccion de resultados de partidos de futbol de La Liga espanola utilizando datos historicos y odds de casas de apuestas.

- **Dataset**: 3,591 partidos, 116 features (10 temporadas)
- **Modelos**: Random Forest, LightGBM, XGBoost
- **Resultado**: F1-Score de 0.57 (modelo In-Play)
- **Tecnologias**: Python, Pandas, Scikit-learn, Optuna

[Ver Proyecto Completo](projects/football-prediction/README.md)

---

### 3. Chatbot RAG para Busqueda de Vehiculos Usados
**Deep Learning | NLP | RAG | LLMs**

Chatbot conversacional que utiliza Retrieval Augmented Generation (RAG) para busqueda semantica de vehiculos usados.

- **Dataset**: 38,531 publicaciones de vehiculos
- **Arquitectura**: RAG con FAISS + OpenAI Embeddings + GPT-4o-mini
- **Caracteristicas**: Busqueda semantica, memoria conversacional, respuestas en lenguaje natural
- **Tecnologias**: Python, LangChain, FAISS, OpenAI API, FastAPI, React

[Ver Proyecto Completo](projects/chatbot-rag/README.md)
